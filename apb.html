付録B OpenCL 1.2 API Reference

重要

本項目ではOpenCL1.2の規格書を著者が意訳・要約したものを掲載します。重複する解説についてはオリジナルの規格書を忠実に意訳しました。

この付録ではOpenCL C言語で使う関数やデータ形のレファレンスを掲載します。付録として掲載するのは、本書はOpenCLを知らないエンジニア向けの解説と、上級者が細かなAPIの仕様・関数を字引する作業を分離するためです。

これからOpenCLを学ぶエンジニアにとっては膨大な情報を与えられても処理できないでしょうが、APIやOpenCL C言語に慣れてきてサンプルのソースコードを解読したい人にはレファレンスがあったほうが望ましいかと思います。

OpenCL 1.2の仕様を網羅したものではないので、必要であれば規格書を読んで頂くことが前提とはなりますが、本書で使う関数については、この付録でカバーします。
