5.1. Maven2

注記

Max OS Xでは、HomeBrewとMac Portの２つのいずれかのパッケージマネージャーをインストールしてください。本書では読者は既にいずれかをインストールしているものと

本項目ではパッケージマネージャーを「Mac Port」としています。

検証環境のMaven.  

$ mvn2 -version
Apache Maven 2.2.1 (r801777; 2009-08-07 04:16:01+0900)
Java version: 1.8.0_05
Java home: /Library/Java/JavaVirtualMachines/jdk1.8.0_05.jdk/Contents/Home/jre
Default locale: ja_JP, platform encoding: UTF-8
OS name: "mac os x" version: "10.9.5" arch: "x86_64" Family: "mac"

参考までに検証環境のMac Portでは以下のようなパッケージをダウンロードできます。

Mac PortのMavenパッケージ.  

$ sudo port search maven
maven1 @1.1 (java, devel)
    A java-based build and project management environment.

maven2 @2.2.1_3 (java, devel)
    A java-based build and project management environment.

maven3 @3.3.9 (java, devel)
    A java-based build and project management environment.

Maven で JNA のライブラリをダウンロードするには以下のように依存性モジュールを記述してください。

pom.xmlの依存性解決.  

<dependencies>
    <dependency>
        <groupId>net.java.dev.jna</groupId>
        <artifactId>jna</artifactId>
        <version>4.2.0</version>
    </dependency>
</dependencies>

Eclipse 等の IDE では編集して保存するだけで依存性が更新されます。
